Front-End Development

Website frontend development involves many skills that work together in order to make a website look incredible, such as:

  • HTML

    HTML is a Markup language that determines how elements are ordered on a website page. This skill is needed in almost all websites that exist today as well as the skill to use this language is a must for any website developer.

  • CSS

    CSS is a language that is the driving force behind a websites design and look. This language also allows for elements on a website to be animated to further improve the visual appeal of the website.

  • JAVASCRIPT

    This language can be used on both the backend and the frontend of a website. With regards to the frontend JAVASCRIPT is used to make the website interactive to the user.

  • BOOTSTRAP JS - FRAMEWORK

    BOOTSTRAP is not a programming language it is a framework that comes with prebuilt components that can be used and edited. These components are designed using a combination of HTML and CSS for the look of the components and use pre-made JAVASCRIPT for the interactivity. Allowing for developers to work faster to reach their required look and functionality.

  • XAML - ASP.Net

    XAML is a language that takes inspiration from HTML as it uses a similar tag structure than HTML however it used used in ASP.Net projects. The Difference between HTML and XAML are that XAML treats layout as part of the markup content whereas HTML and CSS treat it as part of the styling. With XAML, you do the content, some of the functionality and the layout in the C# markup, and do only styling in the style section.

Back-End Development

The backend development involves getting the know how on how you want the website to function as well as the websites interactivity this is the result of using skills and programming languages, such as:

  • JAVA

    Java is an object-oriented programming language that produces software for multiple platforms.

  • JAVASCRIPT

    In order for JAVASCRIPT to fulfill the responsibilities of the backend position, a runtime needs to facilitate the execution of JAVASCRIPT autonomously from the user's browser, operating solely on the server side. A runtime serves as the environment where JAVASCRIPT code is executed. Conventionally, this occurs within a web browser. However, frameworks like Node.js have expanded this capability, enabling JAVASCRIPT to operate on the server side.

  • C#

    C# is pronounced as "C-Sharp." It stands as an object-oriented programming language developed by Microsoft, operating within the .NET Framework. With origins tracing back to the C family, C# shares similarities with widely-used languages such as C++ and Java.

  • PHP

    PHP stands for "PHP: Hypertext Preprocessor." It serves as a popular open-source scripting language, primarily executed on the server.
